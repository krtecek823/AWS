# 🧠 Kiro Puzzle Game

다양한 두뇌 훈련 게임을 제공하는 웹 애플리케이션입니다.

## 🎮 게임 종류

### 1. 🧩 Kiro 퍼즐
- 4x4 그리드에 퍼즐 조각을 맞추는 게임
- 드래그 앤 드롭으로 조각 배치
- 8가지 랜덤 패턴
- 힌트 및 정답 보기 기능

### 2. 🧠 카드 매칭 게임
- 8쌍의 이모지 카드 매칭
- 기억력 향상 훈련
- 이동 횟수 및 매칭 수 표시

### 3. 🔢 숫자 기억 게임
- 순서대로 나타나는 숫자 기억
- 레벨별로 숫자 개수 증가
- 단기 기억력 훈련

### 4. 🧮 빠른 계산 게임
- 60초 제한시간 내 계산 문제 해결
- 덧셈, 뺄셈, 곱셈 문제
- 집중력 및 계산 능력 향상

### 5. 🎨 색상 인식 게임
- 스트룹 효과 훈련
- 글자가 아닌 색상 선택
- 인지 유연성 향상

## 🚀 시작하기

### 필요 조건
- Node.js 16.0 이상
- npm 또는 yarn

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 🛠 기술 스택

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Icons**: Lucide React

## 📱 기능

- 반응형 디자인
- 점수 시스템
- 누적 점수 추적
- 게임별 통계
- 직관적인 UI/UX

## 🎯 게임 특징

- **교육적**: 다양한 인지 능력 향상
- **재미있는**: 게임화된 학습 경험
- **접근성**: 웹 브라우저에서 바로 실행
- **진행 추적**: 점수 및 성과 기록
